# Retail Analytics Dashboard (Power BI)

This repository contains an interactive **Retail Analytics Dashboard** built in Power BI.  
It provides insights into product performance, customer behavior, and regional sales trends.

## 🔑 Features
- **Home Dashboard** – Key KPIs and overall retail performance.
- **Product Analysis** – Top products, sales by subcategory, discount vs profit scatter plot, brand vs sales matrix.
- **Customer Analysis** – Segment distribution, customers by state, gender vs orders.
- **Region Analysis** – Regional monthly sales trends, region vs category comparison, filled map of India with shaded regions.

## 🎨 Design Highlights
- Dark‑neon theme for modern visuals.
- Navigation buttons (HOME, PRODUCT ANALYSIS, CUSTOMER ANALYSIS, REGION ANALYSIS) for smooth page switching.
- Consistent color palettes across charts (blue for electronics, green for furniture, red for fashion).
- Interactive slicers for category, region, and date.

## 📊 Tools & Skills Used
- **Power BI Desktop** for dashboard creation.
- **DAX Measures** for custom calculations (Profit, Total Sales, Regional Sales).
- **Data Modeling** with Customers, Products, and Sales tables.
- **Visualization Techniques**: bar charts, treemaps, scatter plots, matrices, filled maps.

## 🚀 Purpose
This project demonstrates how retail businesses can leverage Power BI to:
- Identify top‑performing products and brands.
- Understand customer segments and geographic distribution.
- Track regional sales trends month by month.
- Make data‑driven decisions with interactive dashboards.
